# ⚡ 빠른 요약 (버튼만 딱)

| 기능 | 누르는 순서 |
| --- | --- |
| 📸 사진 업로드 | 홈 `사진 업로드` 또는 하단 `올리기` → 카메라 영역 탭 → 사진 선택 → `올리기 (N장)` |
| ✅ 이름 지정 | `정리하기` → 사진 선택 → `이름 지정하기` → 아이 선택 → `"이름"(으)로 지정하기` |

| ⏭️ 이름 모를 때 넘기기 | 사진 선택 → `이 아이 누구예요?` → `지금은 넘기고 나중에 이름 지정하기 →` |

| ✏️ 이름 변경 요청 | `아이 관리` → 해당 아이 옆 `이름 변경 요청` → 새 이름 입력 → `요청` |
| 💬 의견 보내기 | 홈 → `💬 의견 보내기` → 내용 입력 → `보내기` |
| 🔑 관리자 로그인 | 홈 → `🔑 관리자 로그인` → 비밀번호 입력 → `확인` |
| 📋 요청 승인/거절 | `아이 관리` → 대기 중인 요청 → `✓ 승인` 또는 `✗ 거절` |

# 📖 전체 상세 가이드

---

## 📸 사진 업로드하는 방법

> ℹ️ 업로드 화면으로 가는 방법은 두 가지예요.
> **방법 A** — 홈 화면에서 **사진 업로드** 카드 누르기
> **방법 B** — 하단 탭에서 **올리기** 탭 누르기

📱 *[스크린샷: 홈 화면 — 사진 업로드 카드]*

1. 위 두 가지 방법 중 하나로 업로드 화면으로 이동한다
2. 가운데 카메라 아이콘 영역을 탭한다

📱 *[스크린샷: 업로드 화면 — 사진 선택 전 빈 화면]*

3. 갤러리에서 사진을 선택하거나 카메라로 직접 촬영한다 **(최대 50장)**
4. 선택한 사진이 그리드로 보이면 확인한다
5. 잘못 선택했으면 **+ 다시 선택** 버튼으로 다시 고른다

📱 *[스크린샷: 업로드 화면 — 9장 선택된 그리드 화면]*

6. 파란색 **올리기 (N장)** 버튼을 눌러 업로드한다

---

## 🗂️ 사진 정리하는 방법

> ℹ️ 정리하기 화면은 두 섹션으로 나뉘어요.
> **이름 지정 필요** — 아직 이름이 없는 사진
> **전송 준비 완료** — 이름이 지정된 사진 (드라이브 전송 가능)

📱 *[스크린샷: 정리하기 화면 — 이름 지정 필요 9장]*

### ✅ 이름 지정하기

1. 하단 탭에서 **정리하기** 탭을 누른다
2. 이름 지정할 사진을 탭해서 선택한다 (여러 장이면 **전체 선택** 버튼 활용)

📱 *[스크린샷: 정리하기 화면 — 3장 선택 후 하단 버튼 표시]*

3. 하단에 나타나는 **🐾 이름 지정하기** 버튼을 누른다
4. 아래에서 아이 목록이 올라온다
    - 검색창에 이름을 입력하면 빠르게 찾을 수 있다
    - 여러 마리를 동시에 선택할 수 있다

📱 *[스크린샷: 이름 지정 드로어 — 아이 목록]*

5. 아이 이름을 탭해서 선택한다
6. **"아이이름"(으)로 지정하기** 버튼을 누른다

📱 *[스크린샷: 이름 지정 드로어 — 테스트용 선택된 상태]*

7. 선택한 사진이 **전송 준비 완료** 섹션으로 이동한다

📱 *[스크린샷: 정리하기 화면 — 이름 지정 필요 / 전송 준비 완료 두 섹션]*

---

### ➕ 목록에 없는 아이 이름 추가 요청하기

1. **🐾 이름 지정하기** 버튼을 눌러 아이 목록을 연다
2. 검색해도 없으면 목록 하단 **+ 새 아이 추가 요청** 섹션을 찾는다
3. 추가할 **아이 이름**을 입력한다
4. **요청** 버튼을 누른다

📱 *[스크린샷: 이름 지정 드로어 — 승인 대기 중 노란 배너]*

5. 노란 배너 *"[이름] 승인 대기 중…"* 이 표시된다
6. 관리자가 승인하면 자동으로 아이가 선택된 상태가 된다
    - 승인 전까지는 **지금은 넘기고 나중에 이름 지정하기 →** 버튼으로 넘길 수 있다

---

### 🚀 구글 드라이브로 전송하기

1. **전송 준비 완료** 섹션에서 사진을 선택한다 (**전체 선택** 가능)

📱 *[스크린샷: 정리하기 화면 — 전송 준비 완료 사진 선택 후 드라이브로 보내기 버튼]*

2. 하단 파란색 **🚀 드라이브로 보내기 (N장)** 버튼을 누른다
3. 하단에 진행률 게이지 바가 실시간으로 채워진다
4. 완료되면 해당 사진이 목록에서 사라진다

---

### ⏭️ 이름을 모를 때 넘기기

1. 이름 모르는 사진을 선택한다
2. 하단 **이 아이 누구예요?** 버튼을 누른다
3. 아이 목록이 열리면 **지금은 넘기고 나중에 이름 지정하기 →** 버튼을 누른다
4. 사진이 **이름 확인** 탭으로 이동되어 다른 봉사자가 도움을 줄 수 있다

---

## 🐾 아이 관리 페이지에서 이름 추가 요청하기 / 이름 변경 요청하기

📱 *[스크린샷: 아이 관리 페이지 — 새 아이 추가 요청 섹션]*

1. 하단 탭에서 **아이 관리** 탭을 누른다
2. 상단 **새 아이 추가 요청** 섹션에서 이름을 입력한다
    - 왼쪽: 내 이름 (선택사항)
    - 오른쪽: 추가할 아이 이름
3. **요청** 버튼을 누른다
4. 관리자 승인 후 아이 목록에 자동으로 추가된다

---

### ✏️ 이름 변경 요청하기

📱 *[스크린샷: 아이 관리 페이지 — 아이 목록과 이름 변경 요청 버튼]*

1. 하단 탭에서 **아이 관리** 탭을 누른다
2. 검색창에서 이름을 바꾸고 싶은 아이를 찾는다
3. 해당 아이 오른쪽 **이름 변경 요청** 버튼을 누른다
4. 새로 바꿀 이름을 입력한다
5. **요청** 버튼을 누른다
6. 관리자가 승인하면 아이 이름이 변경된다

---

## 💬 불편한 점 의견 보내기

📱 *[스크린샷: 홈 화면 — 의견 보내기 버튼]*

1. 하단 탭에서 **홈** 탭을 누른다
2. 맨 아래 **💬 의견 보내기** 버튼을 누른다
3. 팝업 창에서 내용을 입력한다
    - 이름 (선택사항)
    - 의견 내용 (최대 500자)

📱 *[스크린샷: 의견 보내기 팝업 화면]*

4. **보내기** 버튼을 누른다
5. 🎉 화면이 뜨면 전송 완료

---

## 🔑 관리자 기능 사용법

### 로그인

📱 *[스크린샷: 홈 화면 — 관리자 로그인 버튼]*

1. 홈 화면 하단 **🔑 관리자 로그인** 버튼을 누른다
2. 관리자 비밀번호를 입력한다
3. **확인** 버튼을 누른다

📱 *[스크린샷: 홈 화면 — 관리자 로그인 완료 후 화면]*

4. 로그인 성공 시 홈이 관리자 모드로 바뀐다

---

### 📋 요청 승인·거절

📱 *[스크린샷: 아이 관리 관리자 모드 — 대기 중인 요청 2건]*

1. 홈 화면 **승인 요청 N건 대기 중** 배너를 누르거나 **아이 관리** 탭으로 이동한다
2. 화면 상단 **대기 중인 요청** 섹션에서 요청 목록을 확인한다
    - 유사한 이름이 이미 있으면 빨간 경고 문구가 표시된다
3. **✓ 승인** — 아이 목록에 즉시 추가된다
4. **✗ 거절** — 요청이 삭제된다

---

### ➕ 아이 직접 추가

1. **아이 관리** 탭에서 **+ 새 아이 직접 추가** 섹션을 찾는다
2. 아이 이름을 입력하고 **추가** 버튼을 누른다

---

### ✏️ 아이 이름 수정 / 🗑️ 삭제

1. **아이 관리** 탭에서 해당 아이를 찾는다
2. 오른쪽 **✏️** — 이름 수정 후 저장
3. 오른쪽 **🗑️** — 확인 팝업 후 삭제 확정

---

### 🔄 Google Drive 동기화

1. **아이 관리** 탭 상단 **드라이브 동기화** 버튼을 누른다
2. 완료 메시지가 뜨면 동기화 완료

---

### 🚪 로그아웃

1. 홈 화면 또는 **아이 관리** 탭에서 **로그아웃** 버튼을 누른다
