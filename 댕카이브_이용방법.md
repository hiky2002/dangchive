# 🐾 댕카이브 이용 방법

---

## 1. 사진 업로드하는 방법

> 업로드 화면으로 이동하는 방법은 두 가지예요.
> - **방법 A**: 홈 화면에서 **📸 사진 업로드** 카드를 누릅니다
> - **방법 B**: 하단 탭에서 **올리기** 탭을 누릅니다

1. 위 두 가지 방법 중 하나로 업로드 화면으로 이동합니다
2. 가운데 큰 영역(카메라 아이콘)을 탭합니다
3. 갤러리에서 사진을 선택하거나 카메라로 직접 촬영합니다 (최대 50장)
4. 선택한 사진이 그리드로 표시되면 내용을 확인합니다
5. 잘못 선택한 사진이 있으면 **+ 다시 선택** 버튼을 눌러 다시 고릅니다
6. 파란색 **올리기 (N장)** 버튼을 눌러 업로드합니다
7. 업로드가 완료되면 자동으로 정리하기 탭으로 이동합니다

---

## 2. 사진 정리하는 방법

### 📌 기본 화면 구성
- **이름 지정 필요** 섹션: 아직 이름이 지정되지 않은 사진
- **전송 준비 완료** 섹션: 이름이 지정되어 드라이브 전송 가능한 사진

---

### ✅ 이름 지정하는 법

1. 하단 탭에서 **정리하기** 탭을 누릅니다
2. 이름 지정할 사진을 탭해서 선택합니다 (체크 표시가 생깁니다)
   - 여러 장을 한 번에 하려면 **전체 선택** 버튼을 누릅니다
3. 하단에 나타나는 **🐾 이름 지정하기** 버튼을 누릅니다
4. 아이 이름 목록이 아래에서 올라옵니다
   - 검색창에 이름을 입력해 빠르게 찾을 수 있습니다
5. 해당 아이 이름을 탭하면 선택(체크)됩니다 (여러 마리 동시 선택 가능)
6. 파란색 **"아이이름"(으)로 지정하기** 버튼을 누릅니다
7. 선택한 사진들이 **전송 준비 완료** 섹션으로 이동합니다

---

### ➕ 이름이 없을 때 새 아이 추가 요청하는 법

1. **이름 지정하기** 버튼을 눌러 아이 목록을 엽니다
2. 검색해도 아이 이름이 없으면 목록 하단의 **+ 새 아이 추가 요청** 섹션을 찾습니다
3. 빈 칸에 추가할 **아이 이름**을 입력합니다
4. **요청** 버튼을 누릅니다
5. 하단에 노란색 안내 배너가 나타납니다: *"[이름] 승인 대기 중…"*
6. 관리자가 승인하면 자동으로 아이 이름이 선택된 상태가 됩니다
   - 그 전까지는 아래 **지금은 넘기고 나중에 이름 지정하기 →** 버튼으로 넘길 수 있습니다

---

### 🚀 드라이브 전송하는 법

1. **전송 준비 완료** 섹션의 사진을 탭해서 선택합니다
   - 섹션 오른쪽 **전체 선택** 버튼으로 전체 선택도 가능합니다
2. 하단에 나타나는 파란색 **🚀 드라이브로 보내기 (N장)** 버튼을 누릅니다
3. 하단에 진행률 게이지 바가 나타나며 실시간으로 채워집니다
4. 전송이 완료되면 해당 사진들이 목록에서 사라집니다
   - 전송 완료된 사진은 화면 아래쪽 **전송 완료** 섹션에서 확인 가능합니다

---

### ⏭️ 아이 이름을 모를 때 넘기는 법

1. 이름 모르는 사진들을 선택합니다
2. 하단에 나타나는 **이 아이 누구예요?** 버튼을 누릅니다
3. 아이 목록이 열리면 **지금은 넘기고 나중에 이름 지정하기 →** 버튼을 누릅니다
4. 해당 사진은 **이름 확인** 탭으로 이동하여 다른 봉사자가 이름을 지정해 줄 수 있습니다

---

## 3. 아이 관리 페이지에서 이름 생성 요청하는 법

1. 하단 탭에서 **아이 관리** 탭을 누릅니다
2. 상단의 **새 아이 추가 요청** 섹션을 찾습니다
3. 왼쪽 칸에 **내 이름** (선택사항), 오른쪽 칸에 **추가할 아이 이름**을 입력합니다
4. **요청** 버튼을 누릅니다
5. 관리자가 승인하면 아이 목록에 자동으로 추가됩니다

> 이름 검색창에서 기존 아이들을 검색할 수도 있습니다.

---

## 4. 불편한 점 의견 보내는 법

1. 하단 탭에서 **홈** 탭을 누릅니다
2. 메뉴 목록 맨 아래 **💬 의견 보내기** 버튼을 누릅니다
3. 팝업 창이 열리면:
   - **이름** 칸에 이름을 입력합니다 (선택사항)
   - **의견 내용** 칸에 불편한 점, 건의사항 등을 입력합니다 (최대 500자)
4. **보내기** 버튼을 누릅니다
5. 🎉 화면이 뜨면 전송 완료입니다

---

## 5. 관리자 페이지 기능 및 사용법

### 🔑 관리자 로그인

1. **홈** 탭에서 **🔑 관리자 로그인** 버튼을 누릅니다
2. 비밀번호 입력창이 나타나면 관리자 비밀번호를 입력합니다
3. **확인** 버튼을 누릅니다
4. 로그인 성공 시 홈 화면이 관리자 모드로 바뀝니다:
   - **승인 요청 N건 대기 중** 배너가 보이면 대기 중인 요청이 있는 것입니다
   - **관리자 모드 / 로그아웃** 버튼이 표시됩니다

---

### 📋 아이 추가/이름 변경 요청 승인·거절

1. 홈 화면에서 **승인 요청 N건 대기 중** 배너를 누릅니다 (또는 아이 관리 탭으로 이동)
2. **아이 관리** 탭에서 관리자 모드로 진입하면 **대기 중인 요청** 섹션이 표시됩니다
3. 각 요청 카드에:
   - 요청 내용과 요청자 이름이 표시됩니다
   - 유사한 이름이 이미 존재하면 빨간색 경고 문구가 표시됩니다
4. **✓ 승인** 버튼: 누르면 아이 목록에 즉시 추가됩니다
5. **✗ 거절** 버튼: 누르면 요청이 삭제됩니다

---

### ➕ 아이 직접 추가

1. **아이 관리** 탭 → 관리자 모드에서 **+ 새 아이 직접 추가** 섹션을 찾습니다
2. 아이 이름을 입력합니다
3. **추가** 버튼을 누릅니다
4. 아이 목록에 즉시 추가됩니다

---

### ✏️ 아이 이름 수정

1. **아이 관리** 탭 → 아이 목록에서 이름을 바꿀 아이를 찾습니다
2. 해당 아이 오른쪽의 **✏️ (연필 아이콘)** 을 누릅니다
3. 새 이름을 입력합니다
4. 확인 버튼을 눌러 저장합니다

---

### 🗑️ 아이 삭제

1. **아이 관리** 탭 → 아이 목록에서 삭제할 아이를 찾습니다
2. 해당 아이 오른쪽의 **🗑️ (휴지통 아이콘)** 을 누릅니다
3. 확인 팝업이 뜨면 삭제를 확정합니다

---

### 🔄 Google Drive 동기화

1. **아이 관리** 탭 → 관리자 모드에서 상단의 **드라이브 동기화** 버튼을 누릅니다
2. Drive에 있는 아이 폴더 정보와 DB를 동기화합니다
3. 완료 메시지가 뜨면 동기화 완료입니다

---

### 🚪 관리자 로그아웃

1. 홈 화면 또는 아이 관리 탭에서 **로그아웃** 버튼을 누릅니다
2. 화면이 봉사자 모드로 돌아옵니다
